Summary: Input injection risk into PKGBUILD via unsanitized script arguments

Details:

The script `get_latest_version.sh` downloads and parses checksum files from `https://ftp.mozilla.org/pub/firefox/nightly`, which is Mozilla's legitimate server, indicating a valid upstream source. This creates a supply-chain vulnerability: if the connection is intercepted (e.g., via MITM), an attacker could inject a malicious `_version` or `_build_id_raw`, leading to version spoofing or misconfiguration.
