- Exfiltrate data to remote servers
- Modify or delete server files

The explicit warning in the help text confirms this is user-accessible functionality.

Additionally, the use of `eval()` extends to configuration handling:
```python
config[first][second] = eval(arg[1])
```
This allows any user who can modify configuration (admins) to execute arbitrary code during config changes.
</details>
